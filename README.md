# Jogo-mobile-endless-climb
# Dao Ascension: Path to Immortality

## Visão Geral

Dao Ascension: Path to Immortality é um jogo mobile endless de escalada baseado no jogo Getting over it, ambientado em um universo Wuxia.

O jogador controla um cultivador que busca alcançar a imortalidade escalando uma Montanha Celestial infinita utilizando apenas técnicas marciais e habilidades espirituais.

O jogo combina desafio técnico, progressão permanente e competição global através de leaderboards.

---

# 1. Conceito e Proposta de Valor

## Ideia Central

O jogador controla um artista marcial em treinamento que precisa subir a Montanha Celestial para atingir a imortalidade.

A escalada é feita exclusivamente através de:

- Saltos marciais
- Corridas na parede
- Impulsos de Qi
- Técnicas Wuxia

O mapa é procedural e infinito, representando o caminho interminável do cultivo.

---

## Diferencial

- Tema Wuxia com foco real em artes marciais
- Sistema de técnicas desbloqueáveis
- Leaderboards globais competitivos
- Sistema estratégico de Safe Point comprável(f2p)
- Progressão permanente baseada em cultivo

---

## Experiência Principal

O jogador experimenta:

- Alta tensão ao escalar
- Competição por ranking global
- Sensação de evolução do personagem

# 4. Estratégia de Monetização

## Modelo Híbrido

### Anúncios

- Rewarded Ad para:
  - Reviver após queda
  - Ativar técnica temporária

---

### Compras Internas (IAP)

- Jade Espiritual (moeda premium)
- Skins de cultivador
- Remoção de anúncios
- Passe de temporada competitivo

---

### Item Estratégico: Safe Point

O jogador pode comprar um item especial chamado "Talismã de Ancoragem".

Função:

- Permite salvar uma altura específica como ponto de retorno.
- Pode ser usado uma única vez por tentativa.
- Dura x dias
---

## Justificativa

- Leaderboards incentivam competitividade.
- Safe Point reduz abandono precoce.
- Técnicas desbloqueáveis incentivam progressão.
- Monetização não bloqueia jogador gratuito.

- Bater recorde pessoal
- Subir no leaderboard global
- Evoluir estágios de cultivo

## Leaderboards

O jogo possui:

- Ranking global
- Ranking semanal

## Loop Principal

1. Escalar a montanha
2. Cair
3. Receber pontos de cultivo com base na altura
4. Desbloquear técnicas
5. Evoluir estágio
6. Competir por melhor ranking

# 6. MVP (Minimum Viable Product)

## Versão Mínima Jogável

- 1 personagem cultivador
- 1 montanha procedural simples
- Sistema básico de salto e impulso
- 3 técnicas desbloqueáveis
- Sistema de altura e pontos de cultivo
- Leaderboard global simples
- Safe Point
- Rewarded Ad para reviver
